# Travel Companion App
## Created for SIT305 Task 2.1P
### By Isaac McKissack - 218250292

App allows the conversion of 5 currencies, some distances, and temperatures

List of convertible units stored within conversion_array.xml

Addition of further convertible units must occur at the end of the file, as order is integral to current functionality

Conversions handled by calculatorSwitch() function within MainActivity.java

Layout handled in activity_main.xml
